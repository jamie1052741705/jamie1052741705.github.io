Android studio插件整理

1.GsonFormat

快速将json字符串转换成一个Java Bean，免去我们根据json数据手写对应JavaBean。

使用方法：快捷键Alt+S也可以使用Alt+Insert选择GsonFormat

2.Android ButterKnife Zelezny

配合ButterKnife实现注解，从此不用写findViewById。在Activity，Fragment，Adapter中选中布局xml的资源id自动生成butterknife注解。

使用方法：如图所示,在R.layout.XXXX上按Alt+Insert

3.Android Code Generator

根据布局文件快速生成对应的Activity，Fragment，Adapter，Menu。

有感：感觉并不是很实用...

4.Android Parcelable code generator

JavaBean序列化，快速实现Parcelable接口。

5.Lifecycle Sorter

可以根据Activity或者fragment的生命周期对其生命周期方法位置进行先后排序，快捷键Ctrl + alt + K

6.CodeGlance

代码预览小工具，在右侧有代码预览窗口，可以快速的定位代码。

7.JsonOnlineViewer

在Android studio中调试网络接口，能更快速的定位网络接口是否异常。

8.Android Styler

根据layout的Xml文件，生成style代码。







使用方法:

a.复制layout.xml中你要生成Style的代码段。

b.在Style.xml中按Ctrl+Shift+D.

c.根据提示，为你的Style命名.

9.Android Drawable Importer

强大的图片导入插件。它可以导入图标、矢量图、批量导入、多源导入（导入某张图片各种dpi的图片）

10.SelectorChapek for Android

通过资源文件名生成Selector的Drawable.xml文件，经测试，不支持jpg图片格式，支持png图片格式。







命名规格:

11.Android Postfix Completion

可根据后缀快速完成代码。

12.RemoveButterKnife

这是一个可以快速将ButterKnife移除的插件。

12.markdown support

Markdown 语言支持

13.Android-DPI-Calculator

DPI 计算工具，输入1个dpi数值，可以获取其它尺寸的DPI值。

14.Andrpid studio Prettify

可以将代码中的字符串写在string.xml文件中

15.Material Theme UI

添加Material主题到AS，让你的AS更炫...





16. .ignore

.ignore 插件，可以根据选择不同的语言来选择创建不同的.ignore模板，而且还提供了自动补全功能，



17.markdown navigator

MarkDown Navigator 提供markdown 的一些导航工具

18.ECTranslation

android studio翻译插件，可以将英文翻译成中文。


19.AndroidStudioSuperPlugin

最后介绍一个插件集合，这个插件包含Android Studio Prettify，GsonFormat，Android Code Generator，SelectorChapek，Android Parcelable Generator，folding-plugin，Lifecycle-Sorter






Android Studio 插件安装的两种方法

a.在线安装，通过Android studio 的Setting --> Plugins --> Browse Repositories 下载并安装https://plugins.jetbrains.com/?androidstudio提供的插件。

b.本地安装，通过Android studio 的Setting --> Plugins --> Install plugin for disk 选择本地下载好的插件进行安装
